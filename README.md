# 학습플래너 작성 가이드라인


### **<프로젝트의 소개>**

학생(중, 고, 대학생)들의 학습 계획 세우기를 돕는 앱이다. 
하루에 가용가능한 시간을 파악하고, 각 과제를 수행하는데 걸리는 시간을 계산해서 이상적인 계획을 세우는 프로세스를 구현하고자 했다. 
그러나 처음 배우고 프로젝트를 진행하다 보니, 실제 구현 시 모든 기능을 구현하는 데에는 어려움이 있어 날짜 별 투두 리스트 제작까지 구현을 완료하게 되었다.
